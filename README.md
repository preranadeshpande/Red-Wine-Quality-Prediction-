# Red_Wine_Project
 
Factors affecting the quality of the Red Wine are: (Inputs given to the Classifiers)
•	fixed acidity
•	volatile acidity
•	citric acid
•	residual sugar
•	chlorides
•	free sulphur dioxide
•	total sulphur dioxide
•	density
•	pH
•	sulphates
•	alcohol
Quality predicted: (Output from the Classifier: )
•	Range [3,8]
Categorical output: 
•	Poor (quality<5)
•	Good (5<quality<7)
•	Excellent (quality>7)
Binary classes:
•	Bad (quality=3,4,5)
•	Good (quality=6,7,8)
